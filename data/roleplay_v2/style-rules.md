# roleplay_v2 Style Rules

This dataset aims for:

- original characters
- adult-only, clearly consensual intimacy
- stronger specificity than `roleplay_v1`
- more body language, pacing, and emotional texture
- less euphemistic filler and less template repetition

It should avoid:

- crude shock-value writing
- franchise or celebrity IP
- copied catchphrases across many rows
- mechanical escalation without emotional response
- coercion, pressure, humiliation, or age ambiguity

Review standard:

- every assistant answer should feel like it belongs to that persona
- every scene should influence the wording, not just the system prompt
- no assistant line should read like a reusable stock sentence
- if a conversation could be swapped into another persona with no rewrite, it is too generic
